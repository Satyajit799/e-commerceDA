# e-commerceDA
# 📊 Sales Data Analysis using Python

A comprehensive **Sales Data Analysis** project built using **Python, Pandas, Matplotlib, Seaborn, and Plotly**. This project analyzes an e-commerce sales dataset to uncover business insights through data cleaning, exploratory data analysis (EDA), statistical analysis, and interactive visualizations.

---

## 🚀 Project Overview

This project demonstrates an end-to-end data analytics workflow, including:

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Business Performance Analysis
* Customer Analytics
* Sales Trend Analysis
* Profitability Analysis
* Correlation Analysis
* Interactive Executive Dashboard

The objective is to transform raw sales data into actionable business insights that support better decision-making.

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Sales_Data_Analysis.git
```

Navigate to the project folder:

```bash
cd Sales_Data_Analysis
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```
